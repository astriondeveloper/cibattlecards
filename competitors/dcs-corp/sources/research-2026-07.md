# DCS Corp — web research notes (2026-07)

Research conducted 2026-07-17 for the initial battle card draft. All collection
was public-source via web search. Direct page fetches were blocked by the
session's egress proxy (HTTP 403) for most domains, so facts were taken from
search-result content and snippets; each entry notes that. URLs listed are ones
that actually appeared in search results reviewed this session. Card citations
[S#] map to `card.md` Sources; the same numbering is used here.

Method note: no SAM.gov or USAspending.gov API pulls were possible (proxy
denied api.usaspending.gov). FPDS-derived obligations come from the GetWAB
FPDS query-library vendor page surfaced in search results [S4]. A follow-up
session with direct USAspending access should verify the obligations mix and
pull top-contract detail.
